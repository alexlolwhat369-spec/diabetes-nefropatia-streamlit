# 糖尿病肾病照护助手

Aplicación Streamlit en chino simplificado para apoyar a pacientes con diabetes, nefropatía diabética y diálisis semanal.

## Ejecutar localmente

```bash
pip install -r requirements.txt
streamlit run app.py --server.address 127.0.0.1
```

## Publicar en Streamlit Community Cloud

1. Sube esta carpeta a un repositorio de GitHub.
2. Entra a Streamlit Community Cloud.
3. Crea una nueva app.
4. Selecciona el repositorio.
5. Main file path: `app.py`.
6. Deploy.

## Nota médica

Esta app es una herramienta de apoyo y registro. No reemplaza indicaciones de médicos, enfermería o nutrición renal.
